# AI 家教-高中数学-解析几何 Prompt 设计方案

笔试题目交付:面向高中数学解析几何场景的 AI 家教 Prompt 设计,含三档学生分层(普通生/尖子生/冲清北生)、5×5×3 策略立方体,以及 2020 全国 I 卷理科第 20 题的实测对话与可交互动态演示。

## 文件

- `index.html` — 主文档(三个模块的完整内容)
- `demo.html` — 2020 全国 I 卷 20 题的可交互动态几何演示(主文档通过 iframe 嵌入)
