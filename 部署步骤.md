# 部署到 GitHub Pages — 4 步

部署文件夹:这个 `github-deploy/` 目录里有三个文件 `index.html`、`demo.html`、`README.md`。

---

## 步骤 1:在 GitHub 网页上新建仓库

1. 打开 https://github.com/new
2. Repository name:随便取,比如 `ai-tutor-prompt-demo`
3. 选 **Public**(GitHub Pages 免费版要求公开仓库)
4. 不要勾选 "Add a README file"(我们已经有了)
5. 点 **Create repository**

记下你的仓库地址,长这样:`https://github.com/你的用户名/ai-tutor-prompt-demo`

---

## 步骤 2:在终端把文件推上去

打开终端,把下面这段一次性贴进去(替换两处「你的用户名」和「仓库名」):

```bash
cd "/Users/linpeixuan/Library/Application Support/Claude-3p/local-agent-mode-sessions/5b397342-2b77-4a6f-bd98-fd5dfb696c7c/00000000-0000-4000-8000-000000000001/local_b0d6e0e2-e9c6-4245-a8ef-390e8fea8bd3/outputs/github-deploy"

git init
git add .
git commit -m "Initial deploy"
git branch -M main
git remote add origin https://github.com/你的用户名/仓库名.git
git push -u origin main
```

如果是第一次用 git,推送时会要求登录。推荐用 GitHub 网页登录后生成 Personal Access Token(Settings → Developer settings → Personal access tokens),把 token 当成密码贴进去。

---

## 步骤 3:开启 GitHub Pages

1. 进入仓库 → **Settings** → 左侧 **Pages**
2. **Source** 选 `Deploy from a branch`
3. **Branch** 选 `main` / `/(root)`,点 **Save**
4. 等 1-2 分钟,页面顶部会出现绿色提示:`Your site is live at https://你的用户名.github.io/仓库名/`

---

## 步骤 4:打开检查

访问 `https://你的用户名.github.io/仓库名/`,应该看到完整文档。滚到模块三 §3.1 应该能直接拖动滑块,看 CD 直线扫过 (3/2, 0)。

---

## 备选:不想用命令行

GitHub 仓库新建好后,在仓库主页直接拖拽 `index.html`、`demo.html`、`README.md` 到页面上(看到提示 "drop to upload"),点底下的 Commit changes 即可。然后做步骤 3。
